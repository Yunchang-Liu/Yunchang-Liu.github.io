4. Feature Engineering

添加新特征

将具有明显层级关系的分类型属性转换为数字编码

例如数据集中的’KitchenQual’，即厨房情况，’Ex’，’Gd’，’TA’，’Fa’，’Po’分别代表’Excellent’，’Good’，’Average/Typical’，’Fair’，’Poor’。具有明显的层级关系，例如Excellent说明厨房条件特别好，Poor说明厨房条件特别差，和房价具有很强的相关性。

  1. 将X和X_test合并，方便处理
  2. 查看各分类型特征有多少种不同的取值
  3. 分析出具有层级关系的特征
  4. 替换成数字编码

先填充再编码，或者将缺失值编码为’0’

# 先填充再编码，或者将缺失值编码为'0'
df.loc[:,ordinal_cols] = df.loc[:,ordinal_cols].fillna(method='backfill')

# 1] Columns with similar ordered categories [Poor<Fair<Typical/Average<Good<Excellent]

7. Model Creation

预处理 (numerical & categorical)

使用Pipeline和ColumnTransformer进行数据的预处理。

# Preprocessing for numerical data
numerical_transformer = SimpleImputer(strategy='median')

# Preprocessing for categorical data
categorical_transformer = Pipeline(steps=[
    ('imputer', SimpleImputer(strategy='most_frequent')),
    ('onehot', OneHotEncoder(handle_unknown='ignore'))
])

# Bundle preprocessing for numerical and categorical data
preprocessor = ColumnTransformer(
    transformers=[
        ('num', numerical_transformer, numerical_cols),
        ('cat', categorical_transformer, categorical_cols)
    ])

填充缺失值时有三种方式可以选择：

  1. strategy=’median’
  2. strategy=’most_frequent’
  3. strategy=’mean’

分类型特征编码也有多种选择：

  1. 序号编码 Ordinal Encoding
  2. 独热编码 One-hot Encoding（常用）

使用Pipeline

# Create object of class XGBRegressor
xgb = XGBRegressor()

# Bundle preprocessing and modeling code in a pipeline
regressor = Pipeline(steps=[
    ('preprocessor', preprocessor),
    ('model', xgb)
])
